## Switch 开关

### Props

| 参数     | 说明                 | 类型     | 可选值     | 默认值 |
| -------- | -------------------- | -------- | ---------- | ------ |
| v-model  | 开关选中状态         | Boolean  | true/false | false  |
| disabled | 是否为禁用状态       | Boolean  | true/false | false  |
| size     | 开关尺寸,默认单位rpx | *number* | --         | --     |

### Events

| 事件名 | 说明           |
| ------ | -------------- |
| change | 切换开关时触发 |
